# LMS Website
🔥Frontend with React, Shadcn UI & Tailwind CSS 
🔥Redux for state management 
🔥Backend with Node.js and Express.js 
🔥Database integration using MongoDB 
🔥User authentication and profile management 
Start : npm run dev
